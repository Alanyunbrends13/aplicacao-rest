# aplicacao-rest
Aplicação REST com tabela N-N utilizando banco de dados MariaDB e H2

O modelo do banco de dados está disposto no armazenamento de uma entidade de Usuários, que terão o endereço cadastrado neles
Junto aos usuários, todos eles poderão ter 1 ou mais equipamentos, assim como os Equipamentos poderão ter um ou mais usuários. Seria a idéia de uma lanhouse ou uma empresa com equipamentos compartilhados, na qual os usuários podem utilizar os equipamentos de demais colegas
Ou também, poderá servir de histórico de utilização de equipamentos, já que a tabela N-N poderá armazenar estes dados
